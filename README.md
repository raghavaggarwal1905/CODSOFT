# 🔐 Advanced Password Generator GUI

A Python Tkinter GUI application to generate secure passwords with customizable options.

## 💡 Features
- Select password length
- Include/exclude:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Copy generated password to clipboard
- User-friendly interface
